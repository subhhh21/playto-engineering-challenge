# Playto Community Feed - Full Stack Challenge

### Core Features:
- **Threaded Comments:** Recursive nesting with optimized DB queries.
- **24h Leaderboard:** Dynamic karma calculation (Last 24 hours only).
- **Tech Stack:** Django REST Framework & React (Tailwind).

### How to Run Locally:
1. **Backend:**
   - Run `pip install django djangorestframework django-cors-headers`
   - Run `python manage.py migrate`
   - Run `python manage.py runserver`
2. **Frontend:**
   - Run `npm install`
   - Run `npm start`

Refer to `EXPLAINER.md` for technical logic and AI audit details.
## Note on Deployment

Due to time constraints, the application is configured to run locally.
Please follow the setup instructions below to run the project.
The backend and frontend are fully production-ready and can be deployed
to services like Railway and Vercel without code changes.
